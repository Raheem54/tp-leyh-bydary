# 🎓 Student Grade Management System

A streamlined, full-stack web application built with **Laravel**. This system allows administrators to manage student records and link them with academic grades through a polished, responsive dashboard. The frontend combines **Vibe Coding** aesthetics with custom **Bootstrap** and **FontAwesome** integrations.

## ✨ Key Features

* **Student Management (CRUD):** Complete control to add, view, and delete student records.
* **Grade Linking:** Seamlessly associate academic performance data with specific students.
* **Admin Dashboard:** A centralized, secure interface for managing all system resources.
* **Vibe-Driven UI:** A custom-styled frontend that prioritizes clean design and user experience.
* **Responsive Design:** Optimized for all devices using **Bootstrap 5**.
* **Interactive UI Elements:** Enhanced visual feedback using **FontAwesome** iconography.

---

## 🛠 Tech Stack

* **Backend:** [Laravel 10](https://laravel.com/) (PHP)
* **Frontend:** Bootstrap 5, Custom CSS, FontAwesome
* **Database:** MySQL
* **Authentication:** Laravel Fortify
* **Architecture:** MVC (Model-View-Controller)

## 🚀 Installation & Setup

Follow these steps to get the project running locally:

### 1. Clone the repository
```bash
git clone [https://github.com/Raheem54/tp-leyh-bydary.git](https://github.com/Raheem54/tp-leyh-bydary.git)
cd tp-leyh-bydary
```

### 2. Install dependencies
```Bash
composer install
npm install && npm run dev
```
### 3. Environment configuration
```Bash
cp .env.example .env
```
### 4. Generate application key
```Bash
php artisan key:generate
```
### 5. Run migrations
```Bash
php artisan migrate
```
### 6. Start the server
```Bash
php artisan serve
```
Visit http://127.0.0.1:8000 in your browser.
